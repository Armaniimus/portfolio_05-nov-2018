Wie ben ik?

Ik ben Armand geboren in juli-1992. Ik hou van; Gamen, Youtube en Programmeren, Met deze 3 dingen kan ik uren bezig zijn zonder mij te vervelen.

Wat kan ik?

Nu ik het eerste jaar van de opleiding "Applicatie en Media ontwikkeling" heb afgerond, Heb ik al ervaring opgedaan in de volgende programmeer talen.

  • HTML5: Ik heb goed kennis genomen van html5 en kan de rest opzoeken bijvoorbeeld via w3schools of via teamtreehouse.com.
  • CSS3: CSS3 heb ik behandelt ik weet hoe ik webpagina's kan en moet stylen maar ik zou mijn artistiek gevoel graag verder willen ontwikkelen
  • JavaScript: Met JavaScript heb ik veel geoefend en ben hier aardig vaardig in dit is onder andere te zien in 3 van mijn projecten onder andere; rekenmachine, Boter_Kaas_en_eieren en Draughts.
  • PHP: Met PHP heb ik regelmatig geoefent ik heb een kleine libary op git staan met php functies die je kan bekijken.
  • SQL: Met SQL heb ik de kennis om een kleine simpele database te ontwikkelen heb ik. Ook heb verstand van erd's en relationele modellen.

Wat zijn mijn ambities?

Graag wil ik mijn opleiding Applicatie en media ontwikkeling niveau 4 afmaken. Hierna wil ik graag dit vak verder studeren maar dan HBO. En wellicht begin ik daarna voor mezelf of ga ik een universitaire studie doen, maar dit is 7 jaar verder tijd genoeg om er over na te denken.
